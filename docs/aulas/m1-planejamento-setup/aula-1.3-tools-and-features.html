01 — Stack Técnica

As ferramentas por trás do projeto

Cada tecnologia foi escolhida com propósito. Sem excesso, sem modismos — só o que resolve.

Frontend

Next.js 14

Framework React full-stack com App Router, Server Components e Server Actions. Roteamento baseado em arquivos e SSR automático.

App Router Server Components Server Actions SSR
Frontend

React 18

Biblioteca de UI com Hooks, Context API e Suspense. Renderização eficiente com reconciliação virtual.

Hooks Context Suspense
Frontend

TypeScript

Superset tipado de JavaScript. Type safety de ponta a ponta, do banco até o componente, com autocomplete e refactoring.

Static Types Generics End-to-End
Frontend

Tailwind CSS

Framework utility-first para estilização rápida e consistente. Classes atômicas direto no JSX, sem CSS files separados.

Utility-First CSS Variables Responsive
Frontend

shadcn/ui

Coleção de componentes baseada em Radix UI + Tailwind. Código-fonte copiado para o projeto — customizável, acessível e bonito.

Radix UI Acessibilidade Theming
Backend

Supabase

Plataforma backend completa: PostgreSQL, autenticação, Row Level Security, Edge Functions e storage. O backend inteiro em um só lugar.

PostgreSQL Auth RLS Edge Functions Realtime
Pagamentos

Stripe

Infra de pagamentos para SaaS. Checkout sessions, webhooks para eventos, assinaturas recorrentes e gestão de planos.

Checkout Webhooks Subscriptions Customer Portal

Resend

API moderna de envio de emails transacionais. Templates feitos com React Email — emails bonitos com JSX, não com HTML arcaico.

React Email Templates JSX Transactional

03 — Features

O que vamos construir

Cada feature é uma peça funcional do CRM. Sem features decorativas — tudo resolve um problema real.

Pipeline Kanban

Quadro visual com drag & drop entre 6 etapas: Novo Lead, Contactado, Proposta, Negociação, Ganho e Perdido.

@dnd-kit · Supabase

Gestão de Leads

CRUD completo com busca, filtros por status, perfil detalhado e timeline de atividades vinculadas.

React · Supabase · TypeScript

Dashboard & Métricas

Painel com taxa de conversão, funil de vendas, trends semanais e deals próximos do fechamento.

Recharts · Server Components

Multi-tenant

Workspaces isolados com Row Level Security. Cada equipe vê apenas seus dados. Convites por email.

Supabase RLS · Policies

Planos & Billing

Free (2 membros, 50 leads) e Pro (R$49/mês, ilimitado). Checkout, portal do cliente e webhooks Stripe.

Stripe · Webhooks

Sistema de Atividades

Registro de ligações, emails, reuniões e notas. Timeline cronológica vinculada a leads e deals.

CRUD · Timeline UI

Autenticação

Login, signup, session management e proteção de rotas com middleware. Supabase Auth integrado.

Supabase Auth · Middleware

Row Level Security

Segurança a nível de linha no PostgreSQL. Policies garantem que nenhum usuário acessa dados de outro workspace.

PostgreSQL · Policies

Onboarding Flow

Fluxo guiado de criação de workspace após o primeiro login. Nome do time e setup inicial.

UX · Server Actions

Landing Page

Página de conversão com cases de sucesso, features e CTA. Design editorial com identidade PipeFlow.

Next.js · Tailwind · SSG
