Os prompts da aula

Landing page do zero ao commit

Prompt 01 — Planejamento

Criar a branch e alinhar com o Claude o que vai ser construído.

Leia o CLAUDE.md e o PLAN.md. Estamos na aula 2.6 — Landing Page. Crie uma branch pra essa feature e me descreva o que vai implementar.
Prompt 02 — Construção

O prompt principal — descreve o que precisa aparecer na página.

Crie a página inicial pública do PipeFlow. Quem acessar o site sem estar logado vai ver essa página. Ela precisa ter: - Menu no topo com o logo, links de navegação e botão "Começar grátis" - Seção principal com frase de impacto, descrição curta e dois botões de ação - 4 números de resultado (+47% conversão, 3.2x leads qualificados, -62% ciclo de venda, 1200+ times) - Lista das principais funcionalidades do CRM (6 no total) - Tabela de preços: plano Grátis (2 membros, 50 leads) e plano Pro (R$49/mês, ilimitado) - Botão final de chamada para ação e rodapé com o logo - Use as cores, fontes e animações que estão documentadas no CLAUDE.md
Prompt 03 — Revisão

Checar se tudo está funcionando antes de salvar.

Revise a landing page que você criou. Verifica se todas as seções aparecem na tela, se os botões estão visíveis e funcionando, e se a página fica boa no celular também.
Prompt 04 — Commit e finalização

Salvar o trabalho, encerrar a branch e marcar a aula como concluída.

Faz o commit com tudo que foi criado, abre um PR pra main, faz o merge, deleta a branch e atualiza o PLAN.md marcando os itens da landing page como concluídos.
