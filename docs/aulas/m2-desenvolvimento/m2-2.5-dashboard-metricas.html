Os prompts da aula

Dashboard em 3 prompts

Prompt 01 — Planejamento

Branch e alinhamento.

Leia o CLAUDE.md e o PLAN.md. Estamos na aula 2.5 — Dashboard de Métricas UI. Crie uma branch pra essa feature e me descreva o que vai implementar.
Prompt 02 — Construção

Tudo que o dashboard precisa.

Implemente o Dashboard conforme o PLAN.md com dados fake: - 4 cards de métricas: Total de Leads, Negócios Abertos, Valor do Pipeline (R$), Taxa de Conversão (%) - Gráfico de funil mostrando negócios por etapa do pipeline (Recharts) - Tabela com os negócios mais próximos do prazo - Dados mockados mas realistas - Esse é o /dashboard — a primeira tela após o login
Prompt 03 — Revisão e commit

Validar e salvar.

Revise o dashboard. Verifica se os cards mostram valores, se o gráfico renderiza, se a tabela de negócios aparece. Faz o commit, abre PR pra main, faz merge e deleta a branch.
